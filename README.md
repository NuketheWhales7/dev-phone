# dev-phone

## Requirements
- es_extended


###  Usage
Client.lua

### Manually
- Download
- Put it in the `resources` directory
- run SQL file for database

## Installation
- run SQL file to put phone into database

- Add this in your `server.cfg`:
```
start dev-phone
```
